# 15-minute stage demo script

## 0:00-1:30 - Set the frame

"The job is not to make founders aware of Claude. The job is to convert a room full of founders into builders who make a product decision before they leave."

## 1:30-4:00 - Paste a real-ish startup pitch

Use the anonymized AI-reliability sample pitch. Point out the hidden issues: side-window UX, weak MCP tool descriptions, unclear value prop, one-off optimization framing.

## 4:00-7:00 - Run deterministic scoring

Explain that you do not start with magic. You start with a rubric: value prop, urgency, differentiation, platform risk, distribution, and data boundary. Claude improves the answer, but the founder sees the rubric.

## 7:00-10:00 - Generate the founder intervention

Show the rewritten value prop, product wedge, architecture decision, platform-risk answer, and metrics for the week.

## 10:00-12:30 - Show model routing and cost discipline

Explain why not every task deserves the biggest model. Route by ambiguity, risk, context size, and business consequence.

## 12:30-15:00 - Close with activation

"The event is successful only if the founder leaves with a working build, a metric, and a reason to come back next week."
